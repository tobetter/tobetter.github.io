# Installing USB WiFi driver

This page will help you to install USB WiFi driver dongle on Debian/Ubuntu OS.

### RTL8812AU
```bash
$ sudo apt update
$ sudo apt install rtl8821au-dkms linux-firmware
```

### RTL8821AU
```bash
$ sudo apt update
$ sudo apt install rtl8821au-dkms linux-firmware
```

### RTL8821CU
```bash
$ sudo apt update
$ sudo apt install rtl8821cu-dkms linux-firmware
```
